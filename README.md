# README #

## Requirements (WIP) ##
- cask (for Emacs): http://cask.readthedocs.io/en/latest/
`brew install cask`

- virtualenv (for Jedi)
`pip install virtualenv`

- pyflakes (for flymake-python-pyflakes)
`pip install pyflakes`

- autopep8 (for autopep8)
`pip install autopep8`

## Setting
`cd <your cask path>`
`cask install`
