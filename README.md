# README #

## Requirements ##
- [Cask](http://cask.readthedocs.io/en/latest/) (for Emacs)
- Python modules
  - virtualenv (for Jedi)
  - flake8 (for flycheck)
  - yapf (for py-yapf)

## Setting
1. Clone dotfiles
1. Install: `sh install.sh`
1. Install Python modules: `pip install -r requirements.txt`
1. Install [Cask](http://cask.readthedocs.io/en/latest/)
1. `cd ~/.emacs.d`
1. `cask install`
